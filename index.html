###用户基本操作
userdel 删除用户 (参数为需要删除的用户名)
useradd name 创建新用户 用户名为name
passwd 若跟了一个用户名则为修改该用户名密码，若未跟就是修改当前用户密码
su 切换用户 (参数为切换到的用户)
logout / exit 注销
reboot / init 6 重启服务器
shutdown / init 0 关机(非正常关机)
write 给用户发送消息(参数为发送消息的用户名)
wall 向所有人发送消息
mesg y/n 是否接收消息(对管理员的消息无效)

###文件的基本操作
mkdir (make directory) 创建一个文件夹 参数为目录名 (-p 自动创建没用的父文件夹)
touch 创建一个空的文件 参数为文件名
rmdir (remove empty directory) 删除空的文件夹 参数为文件夹名
echo 将内容重定向到指定文件 参数为内容 > 文件名
wget url() 联网通过url获取指定文件
rm 删除文件 参数文件名(-f 强制删除 -R递归删除)
cp 拷贝文件或文件夹
mv 移动文件或文件夹(同一个文件夹下移动作为重命名)
less / more 分屏查看
head / tail 查看文件前文 / 后文
wc 查看文件字符数量和行数
sort 对文本排序-默认升序
nuiq 去除相邻的重复内容
\ 管道 - 连接两个进程(命令)把钱一个进程的输出作为后一个进程的输入
> 输出重定向(>>追加输出重定向)
2> 错误输出重定向
pwd 打印当前工作目录 root的路径为/root 普通用户为/home/用户名
cd 去到那个目录 参数文件路径(.. 一个点当前目录 两个点上级目录)
ls (list directory contents) 参数 -R递归点列出文件夹所有内容 -a 查看所有文件包括隐藏文件
硬连接 相当于给文件增加了一个引用计数(不复制数据的备份)
软连接(符号链接) 相当于时Windows下的快捷方式可以为常用的文件和文件夹在用户的主目录下创建软连接方便操作
ln 参数 源文件 目标文件 (加上-s就是软连接)
inoremap 不要递归(nore)
grep 查找字符串
find 查找文件
coding: utf-8 设置编码
2to3 -w 文件名 将python2文件转换成python3
alias rm=’rm -f’ 别名 将命令重新取个名字可以添加参数
unalias 取消别名

ll -rw-r–r– 1 root root 98 11月 14 11:42 hello.py
第一个字符 d - 文件夹 l - 软连接 - - 文件
第二个到第四个字符:文件的所有者有何种操作权限 r - read - 读 w - write - 写 x - execute - 执行
第五个到第七个字符:文件所有者和同组用户对文件有何种操作权限 r - read - 读 w - write - 写 x - execute - 执行
第八个到底十个字符:其他用户(不是所有者也不同组)的用户对文件有何种操作权限 r - read - 读 w - write - 写 x - execute - 执行

参数 755 为所有用户添加x 执行权限 644为所有用户添加rw 读写权限

数据结构和算法

from collections import deque
deque(maxlen=n)构造函数新建一个大小为n的列队，当新的元素加入并且这个队列已满的时候，最老的元素会被自动移除掉

import heaps
heaps.nlargest(n, nums, key) 在nums中找最大到n个数或者以key为最值寻找
heaps.nsmallest(n, nums, key) 在nums中找最小到n个数或者以key为最值寻找

字典中的键映射多个值

from collections import defaultdict
a = defaultdict(type) 构造一个字典a且值类型为type
a[‘g’].append(1) 向字典a中添加键为g值为1的内容

min = min(zip(dict.values(), dict.keys())) 获取字典dict中的最小值和对应的键
dict = sorted = sorted(zip(dict.values(), dict.keys()))为dict排序
相比用lambda函数来说比较方便

序列中出现次数最多的元素

from collections import Counter
word_counts = Counter(str)
top_three = word_counts.most_common(n)
实例化Counter对象word_counts (字典) 然后调用most_common(n)方法产生一个以元祖为元素的列表元祖内容为出现str序列中n个出现次数最多的元素以及出现总次数
实例化的Count对象的特性是它们可以很容易的跟数学运算操作相结合
